About Megaware Encrypter Program
   
Version : 2.1.0 (fix for meta 2.0)
Programmer : Sandheep G.R
Organization: Megaware
 
This program is intended to serve as a cryptography tool to encrypt text documents and have some very nice algorithms to provide maximum security for the documents. With this you can select any one of the two algorithms namely Algorithm META 1.0 and Algorithm META CBE 1.0 which are described in details in the algorithms section
 
Please consider this program as a beta version. I have not tested the functionality of this program to a large extent. So kindly take the trouble to report any errors to the email gr.sandeep1@gmail.com
