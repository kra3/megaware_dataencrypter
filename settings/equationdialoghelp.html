Equation Dialog Quick Help
 
What is equation Dialog ?
The Equation provides dual security to the code you create. Using the equation dialog you create an equation of values which will be used to encrypt the text. Hence the text will be decrypted only if both the code and the value of the equation is correct. The equation built on the code entered. But beware, same value can be obtained by different equations. So create wise equations.
 
How to create an equation?

The equation can be created using the arithmetic operators and the power function. They are represented by +,-,*,/,^ respectively.Here you enter the operator and then the value to be operated.for eg: To add 10000 you can give +10000. It means that the value 10000 will be added to the code.The format is similar for the substration, multiplication and division. Examples of each is given below.

 
Addition (Operator +)
+10000 - This will added to the code 10000
Substraction (Operator -)
-110 - This will substract value 110 from code
Multiplication (Operator *)
*400 - This will multiply 400 to the code
Division (Operator /)
/30 - This will divide 30 from the code
 
Power function
An exception to the operator function is the power function . In the power function the operand is added similar to other functions as the format, ie operator followed by the value.for eg the power function can be used as ^312. Here the first digit in the value 312 is the base for the operation and the remaining digits are taken as the power. This is same for any value. For ^456 the base is 4 and the power is 56 , ie, the function performed will be 4 raised to 56.
The value of this operation is added to the code.For the power function to be valid, there should be atleast two digits and should not contain more than file digits as operand.
 
Clearing the equation used
If you want to remove a equation you used, you just need to make the equation field empty and click OK on the equation dialog and the equation will be reset .
 
Pit falls in the equation
Must Take Care: If you created an equation for a code and later changed the code, the equation won't change according to the value of new code. But when you edit the equation with new code, the equation will be created with respect to new code.
Though the equation provides the ultimate security. It need to be used with great care. The following need to be taken care during the creation of equation
1) The value obtained after the operation of values from the equation on the code should not be less than 1
This will happen when there are more divide and substraction operations. To avoid this use addition or try decreasing the operand value of addition and multiply operators
2) The value obtained after the operation of values from the equation on the code should not greater than code value multiplied t0 1000
This will happen when there are more multiply and adding operations. For this use less values for addition and multiplication .
3) Error is raised when the operand for the operation is 0
4) The length of operand for the power function should be greater than two digits
5) The operand should not contain any special characters or whitespaces
6) The equation should not contain any other operators other than the listed ones
