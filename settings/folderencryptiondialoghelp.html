Folder Encryption Dialog
What is folder encryption dialog?
The folder encryption dialog is used to encrypt multiple files in a single instance. In this you place all your files to be encrypted in a folder and the folder is selected in the folder encrypter dialog. Now all the files can be encrypted using a single key. The same procedure can be used for the decryption also.
Selecting the folder for encryption or decryption
First copy all the files to be encrypted into a folder and then open the Folder Encryption Dialog. Here click the 'Select Folder' button and from the dialog that pops up , select the folder containing the files. Now the program will list all the files in the list box and you can remove the unwanted files from the list.
Encrypting or decrypting the files
Now to encrypt click on the encrypt button and follow the usual procedure. After encryption the program will place a settings file ENCRYPTIONSETTINGS.FMENCS in the folder which contains the algorithm used, hint and the encrypted files list. This file is used to identify the folder when it is selected for decryption. You can also decrypt all the files independently by opening them in the encrypter.
Getting the details of the encrypted folder
After the encryption, you can read the details of the encrypted folder by clicking the 'Show details of encrypted folder' button and if the encryptionsettings file exists, the details will be shown.
Clearing the values
After encryption or decryption, if you want to start another session, click on the 'Clear all values' button which will clear all the values and clears the listbox and the folder path.
Things to be taken care of
1) Do add a non-text file into the folder to be encrypted.
2) The folder should not contain any subfolders
3) Do not delete the ENCRYPTIONSETTINGS file, Though you can decrypt the files without the encryptionsettings file,it is preferred to keep the file.
4) keep a back up copy of encrypted folder. The wrong decryption code may yield an erraneous result and the decryption cannot be reversed.
